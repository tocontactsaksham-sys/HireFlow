# HireFlow

Advanced Job Portal & Recruitment Platform REST API

Features:
- JWT authentication
- Refresh token rotation
- ADMIN / RECRUITER / CANDIDATE RBAC
- Resume PDF upload
- Email notifications
- Pageable job search
- Caffeine cache
- Profiles dev/prod
- PostgreSQL + Flyway
- Docker Compose
- Swagger OpenAPI
- JPA Auditing
- DTO + MapStruct
